# Tower

Website for [tower.community](https://tower.community) – a startup hub in Holborn, London.

## Structure

```
/                        → Public landing page
/tower-2-confidential/   → Password-protected detailed site
```

## Hosting

Static site deployed via [Cloudflare Pages](https://pages.cloudflare.com). Pushes to `main` trigger automatic deploys.

## Local development

Open `index.html` in a browser – no build step required.
